Boilerplate

HTML5 Boilerplate es una aplicación web que nos proporciona un conjunto de plantillas HTML/CSS/JS para que podamos iniciar el desarrollo de nuestro propio sitio web de un modo rápido, robusto y profesional.

Incluye también un fichero .htaccess, de configuración de Apache que establece algunas reglas de caché para un mejor rendimiento y prepara el sitio para servir video con HTML5

Entre las características más importantes, podemos destacar:

  • Multinavegador (también IE6)
  • Preparado para HTML5
  • Optimizaciones para la visualización en navegadores para móviles
  • Clases específicas para IE, que facilitan un mayor control multinavegador
  • Perfiles Javascript en función del navegador
  • Una hoja de estilos de impresión óptima
  • IE6 pngfix
